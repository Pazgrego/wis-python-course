# Simple Lab Dilution Protocol

### Description
This script calculates the exact volumes needed for a dilution. 
It requires inputs in **ng/µl** and **µl**.

### Sample Input
- **Stock (C1):** 100
- **Target (C2):** 5
- **Final Vol (V2):** 500

### Expected Result
- Stock: 25.00 µl
- Diluent: 475.00 µl
